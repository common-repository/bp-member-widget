A simple widget for members with a search facility.

== Description ==

A simple widget that shows members within a paid/free membership level of your choice. This widget has also a search facility so that users can search either by display name or email address for a member.

Suggestions:

I want this plugin to be useful so if you have any suggestions on how to improve it please let me know. 

== Installation ==

Before you activate this plugin make sure that you have Buddypress v1.2.7 or higher.

1. Upload the plugin to the plugins folder.
2. Go to widgets under design & drag Authors-Widget to the desired area
3. Your Good to go!
